# Kaur Rajbir UI Garden

This is a React + Storybook UI Component Library called `kaur_rajbir_ui_garden`. It contains components like Button, Label, Text, Table, Dropdown, Radio Button, Image, Hero Image, Card, etc.

---

## GitHub Repository

https://github.com/rrajbirkaur2/kaur_rajbir_ui_garden

---

## Prerequisites

- Docker installed on your machine
- Node.js & npm installed (for local development/testing)

---

## Running the App in Docker

# Clone the repository:
cd kaur_rajbirkaur_ui_garden
git status
git add.
git commit -m
git push to main

# Build the Docker image:
docker build -t kaur_rajbir_coding_assignment12 .

# Run the Docker container:
docker run -d -p 8083:6006 --name kaur_rajbir_coding_assignment12 kaur_rajbir_coding_assignment12

# Open your browser and go to:
http://127.0.0.1:8083

# To run Storybook locally for development:
npm install
npm run storybook
